## Optional Documents
If Scott is comfortable and wants better matching, these are useful but not required:
- Recent neurology/ALS clinic summary
- Genetic testing report
- Pulmonary function test / FVC result
- Medication list
- ALSFRS-R score or clinic progression notes
- Prior trial participation records

## Privacy Rule
Default posture: collect the minimum needed, store privately, and share outward only with Scott’s permission.

## Doctor-Facing Output Goal
Turn the above into questions like:
- Based on diagnosis/subtype/timing, are any of these trials realistic?
- Are there genetic tests or respiratory/function metrics needed before eligibility can be assessed?
- Would UCSF or another center recommend contacting the trial coordinator?
- Are expanded-access routes worth exploring?
