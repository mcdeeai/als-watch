# ALS Watch — Trial Fit Review Standard

Every identified trial must answer three practical questions before it is treated as actionable:

1. **Fit** — could this plausibly apply to Scott?
2. **Criteria** — what eligibility/ineligibility gates matter?
3. **Path to consideration** — what is the next real step to be considered?

ALS Watch does not decide eligibility or recommend treatment. It prepares clean, sourced review packets for Scott, Rachel, McDee, doctors, and trial coordinators.

## Required Trial Review Fields

### 1. Trial identity
- Trial title
- NCT ID or source ID
- Source URL
- Sponsor / institution
- Trial status
- Phase
- Study type
- Intervention / treatment
- Locations
- Public contact / coordinator info

### 2. Fit summary
Use one of:
- **Strong possible fit** — worth asking doctor/coordinator soon
- **Possible fit / needs missing info** — promising but requires Scott-specific data
- **Unclear fit** — keep watching or ask doctor only if high-value
- **Likely poor fit** — explain why, do not pursue now
- **Not enough information** — source lacks criteria/contact/location details

### 3. Scott-specific fit checks
Assess only if known; otherwise mark “unknown / ask Scott or doctor.”

- Age range
- Diagnosis type: ALS / suspected ALS / PLS / other motor neuron disease
- Time since symptom onset
- Time since diagnosis
- Site of onset if relevant
- Genetic status: SOD1, C9orf72, FUS, TARDBP, negative, unknown
- For C9orf72-specific leads: exact report wording, pathogenic repeat-expansion status, repeat size/range if listed, and whether genetic counseling has been discussed
- ALSFRS-R score
- FVC/SVC / respiratory status
- Mobility/functional status
- Speech/swallowing status when relevant
- Current ALS meds: riluzole, edaravone, tofersen, etc.
- Prior/current trial participation
- Exclusion risks: ventilator/trach, feeding tube, other major conditions, recent investigational therapy
- Travel feasibility / site access

### 4. Key inclusion criteria
Extract the trial’s major inclusion gates in plain English.

Examples:
- ALS diagnosis by specified criteria
- Age 18–80
- Symptom onset within 24/36/42 months
- FVC/SVC above threshold
- ALSFRS-R above threshold
- Specific genetic mutation required or excluded
- C9orf72-specific requirement: confirmed C9orf72 hexanucleotide repeat expansion or other protocol-defined variant language
- Stable riluzole/edaravone period

### 5. Key exclusion criteria
Extract the obvious disqualifiers.

Examples:
- Too much non-invasive ventilation use
- Tracheostomy / invasive ventilation
- Prior gene/cell therapy
- Recent investigational drug
- Specific mutation excluded
- Severe comorbidities

### 6. Missing information
List exactly what we need before anyone can assess fit.

Example:
- Need symptom onset date
- Need latest FVC/SVC
- Need ALSFRS-R
- Need genetic testing result
- Need exact C9orf72 report wording when a lead is C9orf72-specific
- Need current medication list

### 7. How to get considered
Every actionable trial should include a practical path:

- Ask treating neurologist/ALS clinic about fit
- Ask whether referral is needed
- Contact listed trial coordinator if doctor agrees
- Confirm recruiting status and site availability
- Ask what records/tests are needed for screening
- Ask whether remote pre-screening is possible
- Ask whether UCSF can coordinate or refer if trial is elsewhere

### 8. Draft doctor/coordinator question
Each lead should include a short question block, e.g.:

> Could this trial be clinically realistic for Scott based on his diagnosis, onset timing, respiratory status, genetics, and current medications? If yes, what records or tests would be needed before contacting the coordinator?

For C9orf72-specific leads:

> Does Scott's genetic report match this trial's C9orf72 criteria, and should we ask the coordinator what exact genetic documentation, respiratory/function scores, and records are required for pre-screening?

## Output Format Per Trial

```markdown
## Trial: [Title] ([NCT ID])

**Fit status:** Possible fit / needs missing info  
**Why it matters:** Short sourced summary.  
**Main fit gates:** onset timing, FVC/SVC, ALSFRS-R, genetics, meds, travel.  
**Known fit:** what appears compatible based on known Scott info.  
**Unknowns:** what we still need.  
**Possible blockers:** what may exclude him.  
**How to get considered:** concrete next step.  
**Doctor/coordinator question:** one direct question.  
**Source:** URL
```

## Rule
No trial becomes “actionable” until it has a path to consideration and at least one next question for a doctor or coordinator.
